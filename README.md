# Monitor de Sistema Linux 🖥️

Script en Bash que monitorea en tiempo real los recursos del sistema y genera un reporte automático con fecha y hora.

## ¿Qué monitorea?
- CPU — uso actual por proceso
- RAM — total, usada y disponible
- Disco — uso por partición
- Red — interfaces y direcciones IP
- Procesos — top 5 más activos

## Uso
```bash
chmod +x monitor.sh
./monitor.sh
```
